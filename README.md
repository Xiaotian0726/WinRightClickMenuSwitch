# WinRightClickMenuSwitch

使 Windows 的右键菜单在 Win10/Win11 风格之间切换。

## 风格切换
- 切换至 Win10 风格：运行 `win10style.bat`
- 切换至 Win11 风格：运行 `win11style.bat`
